这篇文章主要是续写我的上一篇Hexo+GitHub Pages搭建个人博客
写作说明：官方文档不够全，如果重新配置主题又得查阅很多文章。这里记录30多项更改，方便查阅。
版本说明：Next主题更新速度还是比较快的，不同版本设置有所不同，请自行区分。我的版本：v5.1.2

文件说明

本文会提到两个配置文件，分别是站点配置文件_config.yml在博客根目录下，还有一个主题配置文件_config.yml在next目录下。请注意区分。

选择主题样式

一种主题，四种样式
打开主题配置文件，找到“Schemes”字段，选择一个样式，去掉前面的#即可。然后hexo s -g在本地调试一下。最好都试一下。

设置语言和网站信息

  • 主题默认英语，在站点配置文件找到language字段，后面添加zh-Hans，冒号后面都要有一个空格。
  • site标签下面的信息都改成自己的。
